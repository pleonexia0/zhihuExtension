# zhihuExtension
知乎扩展插件，提供一键拉黑等功能。
# 主要功能
1.拉黑对某个回答点赞的所有用户  
2.关注对某个回答点赞的所有用户  
3.拉黑某位用户的所有关注者
# 使用方法
打开回答页面，点击图中红圈所示按钮，即可实现各种功能。
